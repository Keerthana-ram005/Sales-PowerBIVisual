# Power BI Sales Dashboard

## Overview
Interactive sales dashboard built using Power BI.

## KPIs
- Total Sales
- Total Profit
- Total Orders
- Average Order Value

## Visualizations
- Sales by Region
- Sales by Category
- Monthly Sales Trend
- Top Customers
- Top Products

## Business Insights
- Technology is the highest revenue-generating category.
- South region is the lowest-performing region.
- Sales peak during year-end months.
- Tamara Chand is the top customer.

## Tools Used
- Power BI
- DAX
- Power Query
